Open using processing (https://processing.org/).

Drag a results file into the window to view information such as:
- Route
- Total
	- Time
	- Emissions
	- Distance
	- Cost
- Bar Chart Summaries

Controls:
- F1 to view routes
- F2 to view Time Chart
- F3 to view Emissions Chart
- F4 to view Distance Chart
- F5 to view Cost Chart

While viewing the routes, you can pan with the mouse buttons and zoom with the mouse wheel.
